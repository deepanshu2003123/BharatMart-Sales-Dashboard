# BharatMart Sales Performance Dashboard 📊

An end-to-end Power BI project built for an Indian retail company **BharatMart** to analyze sales performance across India.

## Problem Statement
Management wanted a dashboard to understand:
- Overall business performance
- Region and city wise sales
- Product profitability
- Year over Year growth
- Most valuable customers

## Dataset
- 3 Tables — Orders, Customers, Products
- 1000 Orders across 3 years (2022-2024)
- Indian Rupee (INR) currency

## Tools Used
- Power BI Desktop
- Power Query (Data Transformation)
- DAX (Data Analysis Expressions)

## Features
- ⭐ Star Schema Data Model
- 📋 4 Dashboard Pages
- 📐 9 DAX Measures
- 🔄 Interactive Slicers

## Dashboard Pages
1. Sales Overview
2. Product Analysis
3. Customer Analysis
4. Time Intelligence

## Key Insights
- North region contributes highest sales
- Technology category leads with 66% of total sales
- Consumer segment generates 49% of revenue
- 48% YOY sales growth from 2022 to 2024
